# Run the Tests

## Text Tests
```sh
make text
```

## Unit Tests
```sh
make unit
```

# Prerequisites

- `make` and a C++ compiler (like `gcc`) is installed on your system and is in the `PATH`
- The `CppUTest` framework is in the directory `CppUTest`

To install `CppUTest` in the right directory, run:

```sh
./install-cpputest.sh
```
